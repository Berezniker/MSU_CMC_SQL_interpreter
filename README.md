# Task10_4

как собирать, используя CMake:  
```
$ mkdir build && cd build
$ cmake .. && make
```

как запускать:  
```
$ ./server_run # запуск сервера
$ ./client_run # запуск клиента
```

как тестировать:
```
$ make test
```

`run_client.cpp` - точка входа в клиент  
`run_server.cpp` - точка входа в сервер
